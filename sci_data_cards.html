Scientific Data Analysis

Learning Outcomes:

Upon completion of this module, students should understand:
  • Choose the most appropriate graph to display data from an investigation.
  • Interpret data from graphs and look for patterns and trends as well as identify outliers.
  • Calculate the mean, median and mode of a data set and determine which most accurately represents the data.
  • Use, read, and write Python code to increase skill in analyzing and interpreting data.
