   Pixel Art I.    


Using the Python library Turtle Graphics to create pixel art.


See the turtle in action


Instructions:

  1. Click the play button on the trinket window below. Trinket is a built in environment to run Python code in the browser and display the graphical output.
  2. Look briefly at the code starting on line 30 and carefully read comments.
  3. On line 41 change the tina the turtle's color to "blue" by changing the line to tina.color("blue"). Run the code again.
  4. On line 45 see the function call to snowflake(). The snowflake() function has one argument which is the size of the snowflake. Change the argument in the parenthesis to 5 and run the code again.
  5. Feel free to play around with code, changing the size background color, turtle color, size of the snowflake etc.




Draw a Square


Instructions:

  1. Click the play button on the trinket window below.
  2. Finish the square by using the forward() function and the left() function.
  3. Find the line where the turtle tess is named. Rename your turtle to something else and run the code. What happens?
  4. You will see red on the screen and and error message saying: "NameError: name 'tess' is not defined on line 11 in main.py", go to line 11 and let's fix it. Python does not know who tess is anymore because you renamed the turtle. Replace tess's first command on line 11 to be a command for your new turtle to move forward by using your turtle's name instead of tess.
  5. Replace the other times that we call give tess a command with your turtle's new name
  6. Finish the square by using the forward() function which moves the turtle forward a certain number of pixels and the left() function which turns the turtle a certain number of degrees.
